## 1. 수집하는 개인정보 항목

본 앱은 별도의 회원가입 없이 이용할 수 있으며, 다음과 같은 정보가 수집될 수 있습니다:

- Firebase Authentication을 통한 로그인 시 이메일 주소 (구글/카카오/네이버 로그인 등)
- 앱 내 기능 이용에 따른 익명화된 이용기록 (예: 점수, 사용일시)
- 오류 발생 시 자동 수집되는 디바이스 정보 및 로그 (Crashlytics 등 활용 시)
